Overview

The Student Performance Predictor is an AI-powered web tool that helps teachers analyze and predict a student's academic progress. It uses data like marks, attendance, and behavior to generate insights and performance predictions.

Why We Built This

Many students struggle silently due to lack of timely intervention. Our goal is to help educators take action earlier using accurate AI insights. This promotes a more personalized and effective teaching approach.

Key Features

  • Predicts performance levels: Excellent, Average, Needs Attention
  • User-friendly dashboard for teachers
  • Easy data upload (CSV format)
  • Improves accuracy with more data over time
